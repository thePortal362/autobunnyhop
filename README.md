# Shedule 1 auto bunnyhop script
### A script to automate bunnyhopping in the game schedule 1, bunnyhopping results in moving faster.
> [!WARNING]
> Do NOT use this script in any games that have an anticheat, the anticheat will detect and block the input and ban you!
> This script is made to work with schedule 1 and probably wont work on most other games.

> [!WARNING]
> This script functions __only__ with Windows!

## How to use:
#### Clone this repo or download the Python (.py) file:
    git clone https://github.com/thePortal362/autobunnyhop.git
#### Open the terminal:
    Right click on the folder you downloaded the script in -> Choose open in terminal
#### Execute the python script:
    python main.py
